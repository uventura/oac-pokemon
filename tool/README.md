# How to use

## Image Conversor

```
pip install Pillow
python3 bmp_img_conversor.py
```

## Image .s Generator

```
g++ ./tool/bmp_bitset.c -o exec
./exec
```

## Audio Conversion
```
pip install mido
python ./tool/translate_midi_riscv.py <path>
## .e.g: python ./tool/translate_midi_riscv.py /../audio/oak-s-lab.mid
```
